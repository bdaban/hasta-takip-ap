# Hasta Takip Sistemi

Bu proje, hastaların randevularını, takiplerini ve tetkiklerini yönetmek için geliştirilmiş bir web uygulamasıdır.

## Özellikler

- Hasta yönetimi (ekleme, düzenleme, silme)
- Randevu takibi
- Hasta takibi
- Tetkik yönetimi
- Doktor yönetimi
- Muayene kayıtları

## Teknolojiler

- Nuxt.js 3
- Vue.js
- Tailwind CSS
- Prisma ORM
- PostgreSQL

## Kurulum

1. Projeyi klonlayın:
```bash
git clone https://github.com/kullaniciadi/hasta-takip-app.git
cd hasta-takip-app
```

2. Bağımlılıkları yükleyin:
```bash
npm install
# veya
yarn install
```

3. Veritabanını oluşturun:
```bash
npx prisma migrate dev
```

4. Uygulamayı başlatın:
```bash
npm run dev
# veya
yarn dev
```

5. Tarayıcınızda `http://localhost:3000` adresine gidin.
